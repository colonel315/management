What is It?

Hey Everybody - Hey (insert name here) is a method that the summer camp I work at uses to get 50+ campers attention at once. So I'm thinking if I can get 50+ campers quiet within 5 seconds using this method, then I can get my 30 students to do it too. Here is a scenerio using it:
*Class is talking to each other about what we are doing that day*
Mr. Burr: Hey everybody!

*Class is now quiet and Mr. Burr can talk to everyone.*
